# 🚀 First Steps with Git and GitHub

*This repository contains exercises to help you learn how to use Git effectively and confidently.*

---

## 📚 Recommended Learning Path

1. **Basics:** Learn to use the Git terminal  
2. **Core Commands:** `pull` → `add` → `commit` → `push`  
3. **Advanced Topics:** `clone` → `branch` → `merge`

---

## 📁 About the Folders

Each folder contains at least two main files:

### `README.md`

- This file provides a brief explanation of the Git commands covered in the folder.
- Includes examples to help you understand how they work.

### `EXERCISE.md` or `EXERCISE.txt`

- This file lists the tasks you need to complete.
- It may also describe specific conditions or scenarios you will encounter during the exercise.

---

## 🧩 Additional Files

- These are files related to the exercises.
- You will **edit**, **delete**, **copy**, **commit**, **push**, or **branch** them during your practice.

⚠️ **Note:** It is your responsibility to keep these files organized and use them as tools to build your Git skills.

---

## ✅ Final Tips

Stay focused, practice regularly, and don't be afraid to make mistakes—  
that's how you'll really learn! 💪
